# EnterpriseWebPortal

業務システム開発を想定した Web ポータルサイトです。  
HTML / CSS を使用して構築しており、事業内容の紹介、開発実績一覧、問い合わせフォームなど、企業向けサイトで一般的な構成を再現しています。

---

## ■ サイト概要

このプロジェクトでは、以下の要素を実装しています。

### ● 事業内容セクション
- 業務システム開発  
- Webアプリケーション開発  
- インフラ・運用支援  

### ● 開発実績（サンプル）
業務システム開発を想定した実績一覧を掲載しています。

- 勤怠管理 Web システム  
- 医療予約管理システム  
- 問い合わせ管理システム  

### ● お問い合わせフォーム
- 名前、メールアドレス、内容を入力する簡易フォーム  
- フロントエンドのみ（送信処理なし）

---

## ■ 使用技術

- HTML5  
- CSS3  
- レスポンシブデザイン（簡易対応）

---

## ■ ファイル構成

```
EnterpriseWebPortal/
 ├─ index.html
 └─ (必要に応じて) css/ images/ js/
```

---

## ■ 制作目的

- 業務システム開発を行う企業サイトの構成を理解する  
- HTML/CSS を用いたフロントエンド実装の基礎を身につける  
- Web 系のポートフォリオとして提出できる形にまとめる  

---

## ■ 備考

本プロジェクトはポートフォリオ用に制作した Web サイトです。
